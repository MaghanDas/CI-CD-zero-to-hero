# Imperative Programming Snippets

## 1) Python: Sum of first `n` numbers
```python
def sum_first_n(n: int) -> int:
    total = 0
    for i in range(1, n + 1):
        total += i
    return total

print(sum_first_n(10))  # 55
```

## 2) C: Find maximum in an array
```c
#include <stdio.h>

int max_in_array(int arr[], int size) {
    int max = arr[0];
    for (int i = 1; i < size; i++) {
        if (arr[i] > max) {
            max = arr[i];
        }
    }
    return max;
}

int main() {
    int values[] = {3, 9, 2, 15, 7};
    int size = sizeof(values) / sizeof(values[0]);
    printf("Max: %d\n", max_in_array(values, size));
    return 0;
}
```

## 3) JavaScript: Count even numbers
```javascript
function countEvens(numbers) {
  let count = 0;
  for (let i = 0; i < numbers.length; i++) {
    if (numbers[i] % 2 === 0) {
      count++;
    }
  }
  return count;
}

console.log(countEvens([1, 2, 3, 4, 5, 6])); // 3
```
